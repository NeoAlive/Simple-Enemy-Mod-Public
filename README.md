## About

Simple Enemy Mod is an addon for TaCZ that introduces armed NPCs into Minecraft, creating dynamic combat encounters and recruitable allies.

This port updates the mod to:

- Neoforge
- Minecraft 1.21.1
- Java 21

along with compatibility fixes and maintenance necessary for newer versions.

***

## Original Features

The gameplay, mechanics and assets remain based on the original project by NekoYuni, including:

- Dynamic soldier encounters
- Recruitable allies
- Commander menu
- Event-based spawning
- Configurable AI
- Resource pack support
- Datapack weapon customization

See the original project for full documentation.
